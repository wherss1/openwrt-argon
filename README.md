# openwrt-argon

新openwrt-argon修改绿色主色调，圆边按钮，圆边图标

删除系统自带的luci-app-argon-config和luci-theme-argon的包，然后把openwrt-argon拉取到package主目录里

后面正常选取编译。

安装完主题编译op系统或安装主题后请清理浏览器缓存，不然很有可能没变化
